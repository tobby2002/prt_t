# prj_t 프로젝트 설명

`/Users/master/prj_t` 아래에 모노레포 형태로 백엔드(uv + FastAPI + Backtrader)와 프론트(Next.js)를 구성했습니다.

## 폴더 구조

- **`backend/`** — `uv` 가상환경, `src/prj_t_backend/`
  - `backtest_engine.py`: 합성 일봉 OHLCV + SMA 크로스 전략, 자산 곡선·체결·샤프·낙폭·거래 통계
  - `main.py`: REST API (`/api/health`, `POST /api/backtests/run`, `GET /api/backtests/{id}`), 메모리에 결과 저장, CORS는 기본 `http://localhost:3000` 등
  - `__main__.py` + `[project.scripts] prj-t-api`: 개발용으로 uvicorn `--reload` 실행
- **`frontend/`** — Next.js 16(App Router) + Tailwind + **recharts**로 자산 곡선, 요약 카드, 거래 테이블

## 실행 방법

### 백엔드 (한 터미널)

```bash
cd /Users/master/prj_t/backend
uv run prj-t-api
```

포트·호스트·reload는 환경변수로 바꿀 수 있습니다.

- `PRJ_T_API_HOST` (기본: `0.0.0.0`)
- `PRJ_T_API_PORT` (기본: `8000`)
- `PRJ_T_API_RELOAD` — `0` 또는 `false`이면 reload 끔 (그 외에는 reload 켬)

### 프론트엔드 (다른 터미널)

```bash
cd /Users/master/prj_t/frontend
npm run dev
```

API 주소를 바꿀 때는 프론트에 `NEXT_PUBLIC_API_URL`(예: `http://127.0.0.1:8000`)을 설정하면 됩니다. 미설정 시 `http://127.0.0.1:8000`을 사용합니다.

## 참고 사항

- 백테스트 데이터는 **외부 시세 없이** 내부에서 만든 일봉 시계열입니다. 실제 심볼·CSV·`yfinance` 등으로 바꾸려면 `backend/src/prj_t_backend/backtest_engine.py`의 데이터 소스만 교체하면 됩니다.
- 프로덕션 빌드 시 Google Fonts에 의존하면 오프라인 환경에서 빌드가 실패할 수 있어, `frontend/src/app/layout.tsx`는 **시스템 폰트**만 사용하도록 되어 있습니다.
- 백엔드 프로세스가 이미 같은 포트에서 실행 중이면 충돌할 수 있으니, 필요 시 기존 프로세스를 종료한 뒤 `uv run prj-t-api`를 다시 실행하세요.
